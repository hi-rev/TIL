# while문
```
while 조건문:
    ~
    ~
```
이것이 기본적인 while문 형식이다.

조건문이 True인 경우에 안의 내용을 계속 반복하게 된다.

```
while True:
    ~
    ~
```
위 처럼 조건문에 True를 넣을 수도 있는데, 이는 안의 내용을 무한루프하게 된다.

break등을 걸어서 무한루프를 탈출할 수 있다.


# 증감연산자
파이썬에는 `i++`과 `i--`같은 증감연산자가 없다.

따라서 증가하거나 감소하는 연산을 하고 싶다면 `i+=1`과 `i-=1`를 사용해야 한다.
